# Cat Interpreter Scoreboard UX & Logic Guide (v4.10.0)

This guide documents the "Gamified Productivity" system used in the Cat Interpreter Assistant.

## 1. Visual Hierarchy (20/80 Rule)
- The Scoreboard occupies the top **20%** of the viewport.
- Transcription/Translation takes the remaining **80%**.
- **Goal**: Minimize distraction while providing high-density status feedback.

## 2. Core Scoreboard Components

### Momentum Bar (`MomentumBar.js`)
- **Ghost Segment**: Shows where you should be *right now* to hit your daily goal.
- **Actual Segment**: Your current progress.
- **Blue Marker**: Survival Floor (5500m Benchmark).
- **Gold Marker**: Growth Target (480m Benchmark).

### Emoji Progress Rows (`EmojiRow.js`)
- **💰 Money**: Shows AR$ earned vs daily target.
- **⏱️ Minutes**: Shows productive minutes vs daily goal.
- **☕ Break**: Shows remaining break budget.
- **🔋 Stamina**: Calculates "Stamina Ratio" (Mins / Break). Elite status is **5.3x**.

## 3. Dynamic Intelligence

### Idle Drift Logic
- When not on a call, a timer tracks **Idle Seconds**.
- A red label (e.g., `-0.5m`) shows how many potential minutes are "drifting away" based on your daily goal.

### Pace Prediction
- Predicts your log-off time based on current momentum.
- Adjusted automatically if you log in late (Shift starts at 9am).

### State Vignettes
- **Green Glow**: Active call (accumulating value).
- **Orange Frame**: On break.
- **Red Frame**: Idle for >45s (Urgency indicator).

## 4. Interaction & HUD Modes

### Condensed vs. Expanded
- **Condensed (123 Mode)**: A 4-column grid for high-density data. Useful during active calls.
- **Expanded (Game Mode)**: Full emoji rows and markers. Best for tracking "The Big Picture".

### Edit Mode (✏️)
- Click the pencil to rearrange grid items or edit daily goals directly via clickable pills.

## 5. CSS & Aesthetics Reference

### Global Design Tokens
- **Base BG**: `#0f172a`
- **Panel BG**: `rgba(30, 41, 59, 0.7)` (Glassmorphism)
- **Success**: `#10b981` (Glows for active states)
- **Warning**: `#fb923c` (Break states)
- **Danger**: `#ef4444` (Idle/Late states)

### Key Animations
- `pulseGlow`: 2s infinite shadow pulse for "In Call" status.
- `shineSweep`: Linear sweep across the progress bars to indicate "Life".
- `coinVacuum`: 720deg rotation and scaling for reward particles.
- `textFloatTarget`: Floating "+AR$" text when a call completes.

### Layout Grid
- **Dashboard Header**: Flex column with `gap: 0.25rem`.
- **Metric Grid**: `display: grid; grid-template-columns: repeat(4, 1fr)`.
- **Viewport Scaling**: Optimized for 900x600 but scales up to 1600px.
