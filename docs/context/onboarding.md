# Onboarding & PWA

## Install

- `manifest.json`: name **CatIntAssist — The Cat's Interpreter Assistant**
- Icon: dog photo (`20180305_210729.jpg`) via PNG + `dog-icon.svg`

## First visit (per device)

1. Welcome bar
2. `?` help flashes 3× — “Click here to learn more”
3. App guide tour
4. Connect button flashes 3× after tour

## Hints

- Double-tap banner (dismissible)
- Idle hint: hold/hover controls when labels hidden
- ⚙️ Settings → show text labels

## Updates

Prebuild writes `version.json` + `sw.js`. Banner: **New app version available — Update**.
