# Scoreboard (TLDR)

See also: [../agents/01_SCOREBOARD.MD](../agents/01_SCOREBOARD.MD)

## Layout

- Max ~20% viewport height (CSS `min(42vh, 320px)` header)
- Progress stack: monthly → step → daily bars
- Toggle 🏷️/🔢 hide labels; hover for tooltips

## Key metrics

- Daily/monthly minutes & AR$
- Pace, breaks, shift late, log-off estimate
- 12-step ladder (1375m steps)

## Modes

- **Numbers grid** vs **game** flip
- Off-call: scoreboard workspace fills `<main id="scoreboard-root">`
