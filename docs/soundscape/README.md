# Soundscape (v4.48.2)

Progressive synthetic audio feedback during calls.

## Synthetic audio engine (no-sample-base)
- **Bag Open**: “purse opening” on call connect
- **Income Ticks**: plays every 60s during a call
- **Denominations**: richer sounds on successive time compounds
  - Diamonds: 20m increments
  - Bills: 5m increments
  - Coins: 1m increments
- **Metal Chest / summary**: banked day summary total chime
- **Call end**: proportional “coin stack crash into purse” payout sound

## Systems
- **Silence Guardian Ping**: inactivity reminder
- **Goal Milestone**: chime when finishing a Dial Goal
- **Smart Mute**: global state prevents synthetic triggers when muted
