# Scoreboard UX Master Specification (v4.10.0)

This document provides a comprehensive breakdown of the Cat Interpreter Assistant Scoreboard—covering its layout, logic, gamification, and aesthetics.

---

## 1. Spatial Architecture
*   **The 20% Viewport Limit**: The scoreboard is strictly capped at 20% of the vertical height. This is a core rule to preserve 80% of the screen for the main transcription and translation feed.
*   **Grid Layout**: Uses a CSS Grid system (`repeat(4, 1fr)`) in condensed mode to ensure metrics are perfectly aligned and legible at any width.

## 2. Visual State System (The Frame)
The app uses a global "Vignette" system that frames the entire window based on the current session state:
*   **Active Call**: A white/green glow (`#10b981`) frames the screen, pulsing gently to indicate value accumulation.
*   **Break Mode**: An orange framing (`#fb923c`) signals the break timer is active.
*   **Idle Alert**: If the user is off-call for >45 seconds, the frame sinks into a deep red framing (`#ef4444`) with a "sinkGlow" animation to encourage picking up a new call.

## 3. The Momentum System
The `MomentumBar` is the primary "At-a-Glance" indicator:
*   **The Ghost Bar**: A semi-transparent white bar showing where you *should* be at this exact minute of your shift to hit your daily goal.
*   **The Progress Cursor**: A bright white vertical line showing your actual position. 
*   **Benchmarks**: 
    *   **Blue Marker**: 5500m (Survival Floor).
    *   **Gold Marker**: 480m (Growth/Legend Target).

## 4. Gamified Metrics (Emoji Rows)
Metrics are visualized using the `EmojiRow` component, which uses fractional emoji rendering:
*   **💰 Earning**: AR$ earned vs Daily Target. Uses `RollingNumber` for satisfying updates.
*   **⏱️ Production**: Productive minutes vs Goal.
*   **☕ Refreshment**: Remaining break time. Emojis change from ☕ to 🍵 as you spend the budget.
*   **🔋 Stamina**: A calculated ratio of `Production / Break`. Elite target is **5.3x**.

## 5. Intelligent Idle Tracking
*   **Drift Logic**: When idle, the scoreboard calculates "Lost Potential". A red label shows exactly how many minutes are subtracted from your "expected profit" every second you remain off-call.
*   **Supportive HUD**: The interface uses "Coaching Tips" instead of negative reinforcement, showing messages like "Standing By" or "Ready for the next one."

## 6. Interaction Design
*   **The Toggle (123 / Emoji)**: Users can switch between "Game Mode" (rich emojis/bars) and "Condensed Mode" (pure numbers/grid) via the `123` button.
*   **Direct Editing**: Clicking on any AR$ or Min "Pill" opens an inline input to modify daily targets without leaving the main view.
*   **Grid Rearrangement (✏️)**: Allows users to drag and drop scoreboard elements to customize their personal dashboard hierarchy.

## 7. Aesthetics & Animations
*   **Glassmorphism**: All panels use `backdrop-filter: blur(12px)` and `rgba(30, 41, 59, 0.7)` for a premium, modern feel.
*   **ShineSweep**: A linear gradient that sweeps across active metrics to make the UI feel "alive".
*   **Coin Rain**: When a call ends, a "Cha-Ching" sound plays, and coin particles fly from the center of the screen toward the total earnings pill using a `coinVacuum` animation.
